# ScreenMe

ScreenMe 是一款专门为带屏小爱音响（LX04）设计的每日大屏，基于 Flutter 开发，支持如下特性：
- 时钟显示
- 本地天气显示（彩云天气）
- 每日 Bing 壁纸
- Apple Health 健康数据显示（消耗卡路里、运动和冥想）
- 工作状态指示（已打卡、未打卡、请假、休假）
- Microsoft TODO 待办事项
